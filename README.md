  This course covers the following material:
</p>

 - Introduction to Compute Canada resources
 - High-Performance Computing (HPC) 
 - Linux
 - Bash scripting
 - Slurm workload manager
 - Running jobs
 - Research data management
 - Data transfer: Globus
 - JupyterHub
 - Hands-on projects
